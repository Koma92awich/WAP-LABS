1. What is the difference between the following 2 statements?
setTimeout(booyah, 2000);
setTimeout(booyah(), 2000);
-The first statement waits for 2000ms before calling the function booyah.
-The second statement calls the function booyah immediately without waiting 2000ms.

4. What is "Unobtrusive Javascript"? What is the practical application of Unobtrusive Javascript (and the reasons for using it)?

The practical application of Unobtrusive Javascript is when we have hundreds of line of code and hundreds of javascript functions , seperating the Javascript code from the HTML and CSS makes it easier to read and work with.
