JSP Introduction Quiz


1. What is the main value of Java Server Pages?

-Allow developers insert java code into HTML by making use of special JSP tgs.

2. How are JSP pages related to servlets?

-When JSP pages are compiled, they are converted to servlets

3. How are JSP pages related to HTML?

-JSP pages add server-side code to an HTML page. Thus JSP tags are withing HTML body

4. Give an example of a JSP scriptlet and show how it will look in the JSP servlet .

-JSP scriptlet: <% count = count *10;%>

-JSP servlet: count = count * 10;%

5. Give an example of a JSP declaration and show how it will look in the JSP servlet.

-JSP Declaration: <%! int count = 0; %>

-JSP Servlet: int count = 0;

6. Give an example of a JSP comment and show how it will look in the JSP servlet.

-JSP comment: <%--This is a JSP comment--%>

-JSP Servlet: The JSP comments are ignored when a JSP is translated

7. Give an example of a JSP expression and show how it will look in the JSP servlet.

JSP Expression: <%= ++count %>

JSP Servlet: out.print( ++count );

8. Give an example of a JSP directive and show how it will look in the JSP servlet.

JSP Page Directive: <%@ page import="java.io.*,java.util.*, javax.servlet.*" %>

JSP servlet: import="java.io.*; import java.util.*; import javax.servlet.*;

9. Explain how an EL expression is evaluated.

The container evaluates the expression by checking in the page scope,request scope, session scope and application scope respective if its not found in any scope. Other wise it ignores the expression
